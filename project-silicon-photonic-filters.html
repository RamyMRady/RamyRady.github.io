Machine-learning-driven control systems for photonic resonators

Overview

Silicon photonic resonators offer exceptional filtering capabilities for RF signal processing, but environmental drift (temperature, voltage) causes frequency instability. Manual tuning is slow and impractical for real-time applications. This project developed machine-learning control algorithms that autonomously stabilize resonators, achieving 10√ó faster tuning and >40dB jammer rejection.

Problem

Traditional silicon photonic filters require constant manual recalibration due to thermal and voltage variations. In high-interference RF environments (e.g., military communications, 5G base stations), maintaining stable center frequencies is critical for blocking out-of-band jammers. Existing approaches relied on slow lookup tables or power-hungry heaters, limiting deployment in dynamic scenarios.

System architecture: RF signal ‚Üí photonic filter ‚Üí detector, with ML-driven feedback control

Key Contributions

  • Reinforcement Learning Algorithm: Developed RL-based controller that learns optimal tuning strategies without prior system modeling
  • Real-Time Adaptation: Sub-millisecond response to environmental changes using lightweight neural network inference
  • Multi-Objective Optimization: Simultaneously optimized center frequency stability, insertion loss, and power consumption
  • Hardware Integration: Implemented on FPGA for low-latency control loop with photonic IC co-design
