# Assets Directory

This directory contains static assets for the website.

## Resume/CV

Please upload your actual CV/resume as `Ramy_Rady_CV.pdf` in this directory.

The current file is a placeholder. Replace it with your actual resume to enable the download functionality across the website.

## File Requirements

- **Filename:** `Ramy_Rady_CV.pdf`
- **Format:** PDF
- **Recommended:** Keep file size under 5MB for optimal loading

## Usage

The resume is linked from:
- Navigation bar (all pages)
- Homepage resume card
- About page download button
